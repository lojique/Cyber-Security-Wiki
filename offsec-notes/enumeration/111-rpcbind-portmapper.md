# 111 - RPCBind/PortMapper

```
# Display a concise list of all registered RPC programs on host
rpcinfo -s 10.11.1.72
# Probe rpcbind on host using version 2 of the rpcbind protocol and display a list of all registered RPC programs
rpcinfo -p 10.11.1.72
rpcinfo domain.local
nmap -sSUC -p111 10.10.10.10
```
